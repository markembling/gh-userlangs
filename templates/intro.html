What is this?

This is a small app which will connect to your GitHub account using the GitHub API and give an overview of the languages you've used. GitHub (and their API) does a fantastic job on a per-repository basis, but there's no way of getting a feel for how it looks across all your repositories.

Get Started

Simply click the button below to log in and authorize this app to use the API on your behalf. It uses OAuth, so you'll see the permissions before granting them and of course you can revoke the authorization at any time in your GitHub account settings.
